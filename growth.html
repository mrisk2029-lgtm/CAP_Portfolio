Growth

This work represents a transition from me disliking the repetitive code.org lessons to fully enjoying them and utilizing them to create a piece of work that I enjoy. I never experimented with Turtle programming until this unit, and even if I was uncertain on how it’d go for me, it ended up turning out amazing. Messing with the features like transparency and colors really brought back the joy of experimentation when using code. And as I started using different programs eventually leading up to HTML, my joy for code has blossomed into a more open-minded perspective on how people can use code in their everyday lives.
